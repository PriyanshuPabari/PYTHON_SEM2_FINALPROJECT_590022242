Vehicle Rental System

A Python-based Vehicle Rental System that allows users to rent vehicles, manage bookings, and track availability efficiently. This project demonstrates basic concepts of programming, file handling, and system design in Python.

📖 Project Description

The Vehicle Rental System is designed to simplify the process of renting vehicles. It enables users to:

View available vehicles
Rent a vehicle
Return a vehicle
Manage rental records

This system can be used as a basic prototype for real-world rental services.

⚙️ Features
🚘 Display available vehicles
📝 Rent a vehicle
🔄 Return a vehicle
📊 Maintain rental records
❌ Prevent renting unavailable vehicles
